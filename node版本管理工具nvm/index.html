node版本管理工具nvm

Posted on 2017-11-24 | Edited on 2020-02-03 | In Node | Comments: | Views:

NVM Github地址

linux ：https://github.com/creationix/nvm

windows: https://github.com/coreybutler/nvm-windows

官方github有详细的安装和使用方法

卸载已安装到全局的 node/npm

如果之前是官网下载安装，或者是brew安装建议，先删除node和全局模块

安装（linux）

通过curl或wget

添加.bashrc或.bash_profile或.zshrc

查看安装成功

如果没有输出，执行

安装node

使用nvm

使用 .nvmrc 文件配置项目所使用的 node 版本

如果默认的版本和要使用的版本不同，但是又不想切换版本，可以使用.nvmrc来指定node

最后

在成功安装之后， node 的真是安装位置是在

会有安装的所有版本， 每个版本中都有自己的npm方便管理和使用

就可以像以往一样使用node了。

####其他

如果 nvm ls-remote 返回 N/A，可以使用非HTTPS的源

如果还是不行可修改nvm.sh文件，在curl加上-k以非加密形式请求。
