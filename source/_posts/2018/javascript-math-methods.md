#### floor()
```
// 向下取整
Math.floor(2.1) // 2
Math.floor(2.6) // 2
Math.floor(-2.6) // -3
```

#### ceil()
```
// 向上取整
Math.ceil(2.1) // 3
Math.ceil(2.6) // 3
Math.ceil(-2.1) // -2

```

#### round()
```
Math.round(3.14) //3
Math.round(3.5) //4
Math.round(-3.14) //-3
Math.round(-3.5) //-3
```
